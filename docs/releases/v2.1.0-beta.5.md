## Highlights

- 内容 metrics 实时刷新：文章、手记、页面、相册详情页在客户端 mount 后拉取最新浏览/点赞/评论数，补偿 ISR 缓存的数字滞后；思考列表页通过批量端点一次刷新当页所有条目。
- 相册交互全链路接入：相册详情页新增浏览追踪、点赞按钮、评论区、底部操作栏和封面背景效果；后端同步支持相册的 analytics/like/comment 指标及 DB 触发器。
- 评论区 WebSocket 实时刷新：当其他用户在当前页面发布新评论时，TanStack Query 缓存自动失效并重新拉取，无需手动刷新。
